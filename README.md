# GenericRecyclerViewAdapter
A Generic RecyclerView Adapter.

You can use this generic recyclerview adapter for any recyclerview in your project. This is a highly efficient adapter. If you had 20-30 recyclerview adapters in your project. With soem good luck all adapters might be reduced to a single adapter without increasing even a single line
